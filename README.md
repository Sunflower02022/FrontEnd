# Sunflower Front-End

## Sobre

O projeto Sunflower é um e-commerce que permite o usuário normal:

- Cadastrar;
- Entrar;
- Selecionar um produto para compra;
- Comprar o produto selecionado;
- além de navegar nas páginas do site.

Permite o usuário administrador:
- Cadastrar;
- Entrar;
- Selecionar um produto para compra;
- Comprar o produto selecionado;
- Cadastrar/editar/apagar produtos;
- Cadastrar/editar/apagar categorias;
- além de navegar nas páginas do site.

O Front-End foi construído com as tecnologias:

- JavaScript;
- Angular;
- TypeScript;
- HTML;
- CSS;
- Bootstrap.

## Como rodar o Front-End

- Clone o repositório usando `git clone git@github.com:sunflower02022:frontend.git`;
- Clique com o botão direito na pasta em que o repositório foi clonado e selecione `Abrir com Code` caso tenha o VS Code;
- Abrir o terminal com as teclas `Ctrl` + `'` ou `Ctrl` + `J`
- Utilizar o comando `npm i` para baixar as depedências;
- Utilizar o comando `ng serve` para rodar o programa em `http://localhost:4200/`.
